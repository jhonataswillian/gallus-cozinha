## 🌐 Instalação e Uso

### **Pré-requisitos:**
- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Editor de código (VS Code recomendado)
- Git instalado

### **1. Clone o Repositório:**
```bash
git clone https://github.com/seuusuario/gallus-cozinha.git
cd gallus-cozinha
```

### **2. Deploy Local:**
```bash
# Abra o arquivo diretamente no navegador
open index.html

# Ou use um servidor local
python -m http.server 8000
# Acesse: http://localhost:8000
```

### **3. Deploy Produção:**
- GitHub Pages configurado automaticamente
- Deploy automático a cada push na branch main

## 🎨 Personalização

### **Alterando Cores:**
No arquivo `assets/css/styles.css`, modifique as variáveis CSS:

```css
:root {
    --primary-color: #B8860B;     /* Dourado principal */
    --secondary-color: #8B4513;   /* Marrom secundário */
    --accent-color: #CD853F;      /* Cor de destaque */
    --gold: #FFD700;              /* Dourado vibrante */
}
```

### **Modificando Conteúdo:**
- **Textos**: Edite diretamente no `index.html`
- **Imagens**: Substitua na pasta `assets/images/`
- **Contatos**: Atualize URLs do WhatsApp, email e Instagram
- **Serviços**: Modifique cards na seção de serviços
